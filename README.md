## 🚀 Featured Projects

### 🤖 [AgentHq](https://github.com/BuildathonAgentHQ/HQ) — AI Orchestration Platform
AI-powered platform coordinating **6 parallel agents** for autonomous code reviews across GitHub repos. Built the safety layer: destructive action interceptor catching dangerous commands across 10+ patterns, multi-agent escalation, budget enforcement, and graceful context fallback using MCP + tree-sitter.

`Python` `FastAPI` `Claude API` `GitHub API` `MCP` `tree-sitter`

### 🛡️ Gated Steering for LLM Safety — Research
Designed a gated steering-vector mechanism with a learned MLP module to control LLM safety behavior at inference time **without modifying base model weights**. Outperformed LoRA fine-tuning on Gemma 3 1B by **+4% on harmful refusal rate** and **+6% on benign accuracy** while training **3x faster**.

`PyTorch` `Transformers` `LoRA` `Gemma 3 1B`

### ⚡ Disaggregated LLM Inference — Benchmarking
Benchmarked **prefill-decode disaggregation** on commodity GCP hardware (NVIDIA L4 GPUs) using vLLM + LMCache. Achieved **~3x improvement in time-to-first-token** under concurrent load. Documented KV cache transfer bottlenecks in heterogeneous GPU setups.

`vLLM` `LMCache` `ZeroMQ` `GCP` `CUDA`

### 🎉 SHaas — Real-Time Social Event Platform
Full-stack platform with microservices architecture, real-time feeds, and event-driven workflows. Split across [Backend](https://github.com/TheFriendRequest/Main-Backend-Service) · [API Gateway](https://github.com/TheFriendRequest/API-Gateway-Service) · [Frontend](https://github.com/TheFriendRequest/frontend-service).

`React` `TypeScript` `FastAPI` `Kafka` `GCP Cloud Run` `WebSockets` `Firebase`

### 💪 [Formly](https://github.com/LogicalVerse/Formly-AI-Gym-Bro) — On-Device AI Fitness Coach
Real-time pose inference on Snapdragon NPU at **30+ FPS** with sub-1s latency, backed by hybrid on-device + cloud Gemini architecture.

`Kotlin` `Qualcomm AI Runtime` `Gemini API` `On-Device LLM`

### 📊 [Portfolio Insight Generator](https://github.com/madhavtibrewal/Portfolio-Insight-Generator)

`Python` `LLMs` `Data Analytics`
